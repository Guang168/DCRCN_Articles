奖励机制大纲

- 翻译提案内容/重要信息 - 礼品 / $25-75 美元等值 DCR
- 翻译重要文章/月报/博客/新闻 - 礼品 / $50-250 美元等值 DCR 
- 翻译内容必须上传Github社区可随意转载

- 原创内容 根据 阅读量，转发量，发布于主流媒体网站 等条件奖励。

后期 ownbit 或 第三方钱包加入 DCR 多签功能可以让捐款多的又愿意参与的负责签名。
